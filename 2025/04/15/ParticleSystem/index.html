ParticlSystem

ParticleSystem

  • Emission 发射模块
  • Shape 发射器形状模块
  • Velocity over Lifetime 生命周期内速度变化模块
  • Limit Velocity over Lifetime 生命周期内速度约束模块
  • Inherit Velocity 继承父对象速度， 粒子速度会受到其父对象的移动的影响
  • Force over Lifetime 生命周期内受力变化模块
  • Color voerLifetime 生命周期内颜色变化模块
  • Color by Speed 颜色受速度的影响模块
  • Size over Lifetime 生命周期内粒子大小变化模块
  • Size by Speed : 粒子大小受速度影响模块
  • Rotation over Lifetime : 生命周期内方向变化模块
  • Rotation by Speed 方向受速度影响模块
  • External Forces 粒子受外力影响模块
  • Noise 粒子受到随机噪声影响模块
  • Collision 碰撞模块
  • Triggers 触发器模块 粒子雨 使粒子不会达到屋子
  • Sub Emitters 子发射器模块 多个粒子系统并行 串行发射粒子
  • Texture Sheet Animation 纹理动画模块 可以控制将一张图分割成多个部分，
    每次将其中的一个部分取出作为粒子贴图
  • Lights 光照模块
  • Trails 拖尾模块 可以给粒子添加拖尾效果
  • Custom Data 自定义模块 为粒子自定义数据
  • Renderer 渲染模块 可以 设置渲染材质球， 拖尾材质球

初始化模块

  • Duration 粒子系统发射粒子的持续时间当开启Looping后， Duration失效
  • Looping 是否循环发射粒子
  • Prewarm 是否开启粒子预热 开启后， 粒子个数从0逐渐怎讲到最大值
  • Start Delay 发射粒子之前的延迟 当开启预热Prewarm后不能使用此项
  • Start Lifetime 粒子的初始生命 (存活时长)秒
  • Start Speed 粒子的初始速度
  • 3D StartSize 粒子初始3D尺寸，开启后需要设置xyz大小
  • Start Rotation 粒子初始旋转角度 3D Start Rotation 开启后此选项消失
  • Randomize Rotation 随机化旋转 取值0-1 表示粒子反向（）旋转的概率
  • Start Color 粒子初始颜色
  • Gravity Modifier 粒子受到重力影响
  • Simulatin Space 粒子运动使用的坐标系 本地坐标系 世界坐标系
  • Simulation Speed 改变粒子系统中所有粒子的速度 取值表示倍数
  • Delta Time 调整使用固定时间还是帧刷新粒子状态
  • Scaling Mode 缩放粒子系统状态
  • Play On Awake 粒子系统启动后是否开启自动播放
