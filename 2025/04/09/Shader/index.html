渲染流水线

  1. 应用阶段 输出渲染图元
  2. 几何阶段 输出屏幕空间的几何信息
  3. 光栅化阶段

应用阶段

  1. 视锥剔除
  2. LOD选择
  3. 设置材质 纹理 Shader
  4. CPU 输出图元 点 线 面
  5. 关键输出： 需要渲染的模型数据 顶点 法线 UV坐标

几何阶段

  1. GUP处理
  2. 顶点着色器 模型空间 -> 裁剪空间 mvp 矩阵变换
  3. 投影裁剪 剔除视锥外的图元 齐次坐标除法 -> NDC坐标
  4. 屏幕映射 NDC坐标 -> 屏幕坐标
  5. 关键输出 屏幕顶点坐标 深度值Z-Buffer 插值后的顶点属性 UV 顶点颜色等等

光栅化阶段

  1. 三角形遍历 生成片元(Fragment) 插值顶点属性 （重心坐标插值UV)
  2. 片元着色器 逐像素着色 （计算光照 采样纹理)
  3. 逐片元操作 深度测试（Z-Test）模板测试 颜色混合（Alpha Blending)
  4. 最终输出 写入帧缓冲（Fram Buffer）生成屏幕像素

CPU和GPU通信

  1. 将数据加载到显存中 将网格和纹理加载到显存中， 等待CPU指导渲染状态
  2. 设置渲染状态
  3. 绘制调用 DrawCall

矩阵

  1. 矩阵和标量的乘法
  2. 矩阵和矩阵的乘法 第一个矩阵的列数必须要和第二个矩阵的行数相同 结果是第一矩阵的行数 第二个矩阵的列数
    对于第一个矩阵使用行矢量 点积 第二个矩阵的列矢量
  3. 方块矩阵 行列相等 对角矩阵 除了对角外其余都是0
  4. 单位矩阵 对角矩阵对角分量都是1 任何矩阵乘以单位矩阵还是原来的矩阵
  5. 转置矩阵 转置矩阵的转置等于原来的矩阵
  6. 逆矩阵 前提: 是一个方阵 原矩阵乘逆矩阵等于单位矩阵 逆矩阵的逆矩阵是原矩阵的本身 单位矩阵的逆矩阵是本身
    转置矩阵的逆矩阵是逆矩阵的转置 矩阵相乘后的逆矩阵等于反向串接各个矩阵的逆矩阵
  7. 正交矩阵 一个矩阵和转置矩阵的乘积是一个单位矩阵, 那么这个矩阵就是正交

顶点的坐标空间变换过程

  1. 模型空间(对象空间) 自身独立的坐标空间 模型空间的坐标轴使用自然方向 原点在模型上
  2. 世界空间 原点在世界中心
  3. 顶点变换的第一步 将顶点坐标从模型空间变换到世界空间(模型变换)
  4. 观察空间 (摄像机空间) 顶点变换第二部 将顶点从世界空间变换到观察空间 观察变换
  5. 齐次裁剪空间 (裁剪矩阵/投影矩阵) 顶点变换第三步 将顶点从观察空间变换到齐次裁剪空间
    空间决定-视锥体(六个平面组成这六个平面被称为裁剪平面)
    视锥体有两种类型 正交投影 透视投影
    近裁剪平面 和 远裁剪平面 决定了摄像机的深度范围
  6. 屏幕空间 真正的投影 将视锥体投影到屏幕空间
    首先进行标准齐次除法(透视除法) 即归一化的设备坐标(NDC) 将齐次空间坐标转换到NDC
  7. 切线空间 法线映射

BRDF光照模型

标准光照模型

  1. 自发光 emissive 给定一个方向 表面向该方向发射多少辐射量
  2. 高光反射 specular 当光源照射到物体表面 表面会完全镜面反射多少辐射量
    Phong模型 基于反射向量
    关键矢量: n 法线矢量 I 光源矢量(入射光) v 观察矢量 r 反射矢量
    r = 2(n * I)n - I
    高光反射 = (光源颜色 * 材质高光反射颜色) max(0, dot(v, r))^gloss(光泽度)

Blinn模型 反光更大 基于半角向量 (该模型是各向同性 即固定视角和光源 旋转这个表面 表面反射不会发生改变) (与之相反向头发可能就需要各向异性) 同时
这两个模型无法使用菲涅尔反射
h 半角向量 = (v + I) / |v + I|
高光反射 = (光源颜色 * 材质高光反射颜色) max(0, dot(n, h))^gloss(光泽度)
3. 漫反射 diffuse 当光源照射到物体表面 该表面会向每个方向散射多少辐射量

  • 兰伯特定律 反射光的强度 和表面法线和光源方向的余弦值成正比
    关键向量 : n 法线单位矢量 I 光源 单位矢量
  • 漫反射= (光源颜色 * 材质漫反射颜色 ) max(0, dot(n, I))
  • 半兰伯特模型
  • 漫反射= (光源颜色 * 材质漫反射颜色 ) (dot(n, I) * 0.5 + 0.5) // 适合风格化
