Shader

渲染流水线

  1. 应用阶段 输出渲染图元
  2. 几何阶段 输出屏幕空间的几何信息
  3. 光栅化阶段

应用阶段

  1. 视锥剔除
  2. LOD选择
  3. 设置材质 纹理 Shader
  4. CPU 输出图元 点 线 面
  5. 关键输出： 需要渲染的模型数据 顶点 法线 UV坐标

几何阶段

  1. GUP处理
  2. 顶点着色器 模型空间 -> 裁剪空间 mvp 矩阵变换
  3. 投影裁剪 剔除视锥外的图元 齐次坐标除法 -> NDC坐标
  4. 屏幕映射 NDC坐标 -> 屏幕坐标
  5. 关键输出 屏幕顶点坐标 深度值Z-Buffer 插值后的顶点属性 UV 顶点颜色等等

光栅化阶段

  1. 三角形遍历 生成片元(Fragment) 插值顶点属性 （重心坐标插值UV)
  2. 片元着色器 逐像素着色 （计算光照 采样纹理)
  3. 逐片元操作 深度测试（Z-Test）模板测试 颜色混合（Alpha Blending)
  4. 最终输出 写入帧缓冲（Fram Buffer）生成屏幕像素

CPU和GPU通信

  1. 将数据加载到显存中 将网格和纹理加载到显存中， 等待CPU指导渲染状态
  2. 设置渲染状态
  3. 绘制调用 DrawCall

顶点着色器

  • 顶点空间变换 输入的每一个顶点都会调用一次顶点着色器
  • 顶点动画 模拟水面 布料
  • 基本工作 : 把顶点坐标从模型空间转换到齐次裁剪空间
  • 顶点着色

曲面细分着色器

  • 细分图元

几何着器

  • 逐图元着

裁剪

  • 剔除相机视野外的顶点 并剔除一些三角图元
  • 图元关系 完全在摄像机视野内 部分在(进行裁剪) 在视野交界处建立新的顶点 完全不在

屏幕映射

  • 将图元的坐标映射到屏幕坐标
  • 屏幕映射得到的屏幕坐标 决定顶点对于哪个像素以及距离这个像素有多远

三角形设置

  • 计算三角形网格数据表示

三角形遍历

  • 检查每个像素是否呗三角形覆盖 使用三角形三个顶点对像素进行插值 => 片元序列(屏幕坐标 深度信息 顶点信息 法线 纹理坐标)

片元着色器

  • 单个片元操作
  • 逐片元着色

逐片元操作

  • 修改颜色 深度缓冲 进行混合
  • 决定每个片元的可见性 模板测试 深度测试
  • 如果一个片元通过了所有的测试 则进行输出合并 颜色于颜色缓冲区进行合并 测试不通过全部舍弃

模板测试

  • 限制渲染的区域 渲染阴影 轮廓渲染

问题

  1. 在3ds Max 中,默认坐标方向是: x轴正方向指向右方, y轴正方向指向前方 z轴正方向 指向上方 它是左手还是右手坐标系
    答: 右手坐标系
  2. 左手坐标系中 坐标(0, 0, 1) 绕y轴正方向 旋转90 旋转后的坐标是? 如果右手坐标系呢?
    答: (-1, 0, 0) (1, 0, 0)
  3. 在Unity 中,新建 场景中 主摄像机位置位于实际额空间中的 (0, 1,, -10)位置,在不改变
    摄像机的任何设置,保持旋转0, 缩放1, 世界空间 (0, 1, 0) 新建一个球体, 在摄像机的观察
    空间下,球体的z值是多少, 在模型空间下,球体的z值是多少
    答: 在摄像机的空间下,z值是-10, 模型空间下,球体的z值是0

点 和矢量

  1. point
  2. 矢量 模(长度) 和方向
  3. 标量 模
  4. 矢量乘法 每个分量乘以标量 表示对矢量v进行一个标量的放大 k小于0 取反 最后为矢量
  5. 矢量除法 每个分量除以标量 矢量只能被标量除
  6. 矢量加减法 每个分量相加减 三角形定则 加法 a的尾连到b的头 减法 a-b a的尾连到b的尾
  7. 矢量的模计算 每个分量的平方求和 然后求平方根
  8. 应用 顶点的法线方向 和光源方向 这里需要使用单位向量 unit vector 即模为1的矢量 归一化 的矢量
  9. 归一化 矢量 除以矢量 的模 在unity urp shader中 TransformObjectToWorldNormal函数的参数doNormalize=true对
    法线进行了归一化,确保光照的正确性,避免缩放的影响

矢量点积

  1. 矢量的点积 内积 dot 结果是一个标量 各个分量相乘最后求和 得出一个标量
  2. 点积的应用 投影 单位矢量 a 不限长度的矢量b 得到b在a方向上的投影
  3. 点积投影 当方向相反即夹角大于90 投影为负数shu 垂直的时候为0 小于90则为正
  4. 点积的性质1 对点击其中一个进行矢量进行缩放 即对点积进行缩放
  5. 点积的性质2 可以对其中一个进行矢量的加减
  6. 点击的性质3 矢量和本身点击 等于模的平方
  7. 余弦值 两个单位矢量的点积
  8. 余弦公式 模相乘 乘以余弦值

矢量叉乘

  1. 叉积的结果还是一个矢量
  2. 正弦公式 模相乘 乘以正弦值
  3. 叉积的应用 计算垂直一个平面,三角形的矢量
